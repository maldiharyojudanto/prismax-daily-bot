# Features
- Auto check-in (10 pts/day)
- Auto chat (50 pts/day)

# Requirement
- VS Code
- Node.js or Bun

# How to get token
- Open discord
- Inspect Element (F12)
- Go to Network -> Fetch/XHR -> search "get-users" and select that -> Response -> copy hash_code value

# How to run
- Open VS Code, open index.js
- Fill datas.txt with your solanaaddress/email|token, example: asd@gmail.com|token
- Open terminal
- Command:
  > npm install

  > node index
- Done
